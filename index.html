CV Studio

Ikuti 4 langkah ini untuk membuat CV profesional siap kirim dalam hitungan menit.

01
Salin Prompt CV
Klik tombol "Salin Prompt" di bawah. Prompt sudah diformat siap pakai — berisi instruksi membuat CV ATS-friendly format A4.
02
Paste ke ChatGPT / AI lain
Buka ChatGPT, Claude, atau Gemini. Paste prompt yang sudah disalin, lalu jalankan. Tunggu AI generate kode HTML CV kamu.
03
Isi Data Pribadimu
Di chat AI, minta isi nama, kontak, pendidikan, pengalaman kerja, dan prestasi kamu. Sesuaikan sampai hasilnya pas.
04
Taro Teks HTML nya ke Sini ↓
Copy kode HTML hasil AI. Buka tool di bawah, paste kode HTML-nya, klik Preview untuk cek tampilan, lalu Download PDF!
