# 📄 CV Studio — Build CV You Own

![Preview](./preview.png)

**CV Studio** adalah web tool minimalis yang dirancang untuk menjembatani output AI (seperti ChatGPT, Claude, atau Gemini) menjadi dokumen CV profesional berstandar ATS-friendly dalam format PDF.

Seringkali AI memberikan kode HTML CV yang bagus, tapi pengguna bingung bagaimana cara mengubahnya menjadi PDF yang rapi dan sesuai ukuran A4. Tool ini menyelesaikan masalah tersebut dengan fitur *Instant Preview* dan *PDF Export*.

---

## ✨ Fitur Utama

- **Prompt-to-CV Workflow** — Prompt khusus untuk hasil HTML optimal dari AI  
- **ATS-Friendly Focus** — Struktur mudah dibaca sistem rekrutmen  
- **Live Preview** — Preview real-time  
- **A4 PDF Export** — Export 1 klik (jsPDF + html2canvas)  
- **Modern UI** — Dark mode clean & responsif  

---

## 🚀 Cara Penggunaan

1. Copy prompt dari web  
2. Paste ke ChatGPT / AI  
3. Generate CV  
4. Paste HTML ke CV Studio  
5. Preview & Download PDF  

---

## 🛠️ Teknologi

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (ES6+)  
- html2canvas  
- jsPDF
